== Description  ==
Sharing WhatsApp  with custom post type.
Sharing WhatsApp  button shown only these devices the decice are Android,iPhone,Windows Phone,Nokia S40,BlackBerry,Nokia S60,BlackBerry 10.
This plugin allows to add post . This plugin creates a custom post type.
The Sharing WhatsApp can be shown on posts, pages using short code [whatsapp_sharing].

= General Features =
Add posts.
manage posts.
Show sharing whatsapp on post or page with short code.

== Installation == 
The automatic plugin installer should work for most people.
Manual installation is easy and takes fewer than five minutes.

Download the plugin, unpack it and upload the 'Sharing WhatsApp' folder to your wp-content/plugins directory.
Activate the plugin through the 'Plugins' menu in WordPress.
Go to WhatsApp Sharing  menu in WordPress Admin. Add post there.
To show post and share on post or page use shortcode: [whatsapp_sharing].

To use the plugin in theme file, use below code:
do_shortcode("[whatsapp_sharing]");
